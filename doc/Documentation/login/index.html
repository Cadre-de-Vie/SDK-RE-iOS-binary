Certains opération sur l'API demandent à l'utilisateur d'être loggé. Ces opérations sont:

  • Editer un utilisateur
  • Avoir sa propre fiche utilisateur
  • Gérer les favoris
  • Gérer les invisibles (c-a-d les Properties que l'utilisateur ne veux plus voir dans ses futures recherches)
  • Gérer les POIs
  • Gérer les projets
  • Gérer les alertes

Création d'un utilisateur

Pour créer un nouvel utilisateur il faut initialiser une instance de CDVUser, puis appeller la méthode createUser() du CDVManager


 let newUser = CDVUser(dictionary: [
    "email": "user@mondomain.com",
    "password": "monpassword",
    "first_name": "JeanMichel"])

    CDVManager.sharedInstance().createUser(newUser) { (error, user) in
    if (error != nil) {
        NSLog("Creation failed Error: \(error!.localizedDescription)")
    } else if user != nil {
        NSLog("User Created \(user!.dictionary())")
    }
}

Les paramètres disponibles sont:

  • username
  • email
  • password
  • civility
  • first_name
  • last_name
  • mobile
  • birth_date
  • locale
  • address_line_1
  • address_line_2
  • zip
  • city

Modification d'un utilisateur


 let updatedUser = CDVUser(dictionary: [
            "id": 70,
            "first_name": "Jean-Edmond"])
        CDVManager.sharedInstance().update(updatedUser) { (error) in
            if (error != nil) {
                NSLog("onUpdateUserButtonClick Error: \(error!.localizedDescription)")
            } else {
                NSLog("User updated")
            }   
        }

Login


 CDVManager.sharedInstance().login(withEmail: "user@mondomain.com", password: "monpassword") { (error, user) in
            if (error != nil) {
                NSLog("Login failed. Error: \(error!.localizedDescription)")
            } else if user != nil {
                NSLog("User \(user!.cdvEmail!) logged with token : \(user!.cdvToken!)")
            }

        }

Récupérer les détails sur l'utilisateur loggé


CDVManager.sharedInstance().me { (error, user) in
    if (error != nil) {
        NSLog("onMeButtonLick Error: \(error!.localizedDescription)")
    } else if user != nil {
        NSLog("My details \(user!.dictionary())")
    }

}
